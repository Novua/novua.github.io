Mineblob

Mineblob is a minecraft server in development providing a unique minigames experience unlike any other server.


With custom coded game modes, cosmetics and unique systems to ensure a one-of-a-kind experinece.

AnimationsCore

AnimationsCore is a Minecraft spigot plugin to work with ModelEngine & MythicMobs to create animations for events such as: Join, Leave, Death etc.


It is maintained for latest versions of minecraft with many new features and events being added regularly with Animation packs being available on MCmodels.net by a variety of vendors.

Project HOG

Project HOG is a Minecraft server in development that takes the style inspired by many mobile games in the Tower Defense genre.


Utilising 3D models and custom plugin technology, crafting a completely new style of minecraft server experience like no other!
