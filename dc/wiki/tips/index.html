Troubleshooting

  • "Unknown sequence/camera id": verify YAML ID and run /directorscut reload.
  • Camera not resolving: check anchor.source/look_at.source exists in current context.
  • Model camera wrong facing: tune model_engine.camera_rotation.yaw_offset and pitch_offset.
  • Model camera missing: confirm ModelEngine plugin installed, model ID valid, bone name exists.
  • QTE not triggering input as expected: ensure selected input matches player action and your server version supports PlayerInputEvent bridge (fallback movement tracking is used otherwise).
  • Sequence exits unexpectedly: inspect endconditions plus any conditionend lines in script.
  • Max uses reached: remove/increase max_uses in sequence settings.
