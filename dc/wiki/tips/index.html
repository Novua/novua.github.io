Tips & Troubleshooting

  • Use /directorscut reload after editing YAML files.
  • Camera definitions are merged by ID — duplicate IDs will override earlier entries.
  • If a camera fails to resolve, validate that the anchor and look-at sources exist in the current context.
  • Use lock_player or freeze_player when you need to keep players from breaking the cinematic.
  • Try the example YAML files from DirectorsCut/examples if you want a working starting point.
