Switch Relativity (Players ↔ Mobs)

Relativity is controlled by the anchor and look_at sources. Swap these sources to follow players, NPCs, or MythicMobs entities.

GoalAnchor / Look AtNotes
Follow the viewer anchor: { type: player, source: self } Ideal for player POV or shoulder cams.
Follow the interacted entity anchor: { type: entity, source: interacted } Use when a player interacts with an NPC or block-based trigger.
Follow a MythicMobs spawn anchor: { type: entity, source: mythicmobs:<id> } The ID comes from earlier spawnmob instructions.
Follow a MythicMobs skill targeter anchor: { type: entity, source: mythicmobs_targeter } Uses the entity targeted by the MythicMobs mechanic.
Use a fixed point in the world anchor: { type: absolute, absolute: { world, x, y, z, yaw, pitch } } Great for static establishing shots.

To switch a shot from a player-relative camera to a mob-relative camera, keep the same positioning block and replace anchor.source (and optionally look_at.source) with the mob target you want.

# Player-relative
anchor: { type: player, source: self }
look_at: { type: player, source: self }
positioning: { mode: relative, distance: 2.5, yaw_offset: 20, y_offset: 1.2, pitch: -5 }

# Mob-relative (switch to spawned MythicMob)
anchor: { type: entity, source: mythicmobs:cinematic_dummy }
look_at: { type: entity, source: mythicmobs:cinematic_dummy }
positioning: { mode: relative, distance: 2.5, yaw_offset: 20, y_offset: 1.2, pitch: -5 }
