Camera Definitions

Camera files are loaded from plugins/DirectorsCut/cameras/*.yml with structure pack + cameras: map.

pack: demo_pack
cameras:
  intro_shot:
    label: Intro Shot
    anchor:
      type: entity
      source: self
    look_at:
      type: entity
      source: self
    positioning:
      mode: relative
      distance: 5.0
      y_offset: 1.8
      yaw_offset: 180
      pitch: 10
    lock_player:
      move: true
      look: false

Target types

  • entity (default): use source such as self, spawned mob alias, or integration target entity.
  • location: resolve from integration target location context.
  • absolute: requires absolute.world/x/y/z/yaw/pitch.
  • none: no source lookup.

Positioning modes

  • relative: offset from anchor (supports distance + yaw_offset + y_offset, or direct distance_x/y/z).
  • absolute: fixed absolute.world/x/y/z/yaw/pitch.

Note: legacy duration_ticks in camera files is ignored; sequence script timing controls transitions now.

In-game creation

  1. Run /directorscut editsequence <sequenceId> or create <id>.
  2. Run /directorscut editsequence makecamera.
  3. Enter camera ID in chat; file is saved as editor-<id>.yml.
