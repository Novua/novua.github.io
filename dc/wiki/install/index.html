Install & Quick Start

  1. Drop the DirectorsCut jar into your server’s plugins/ folder.
  2. Start or reload the server to generate plugins/DirectorsCut.
  3. Copy the example YAML files from DirectorsCut/examples into your plugin data folder.
  4. Run /directorscut reload to load cameras and sequences.
  5. Test with /directorscut play <sequenceId> or enter camera creator mode.

Need details on camera YAML fields? Open camera_settings.html at the repo root for a full reference.
