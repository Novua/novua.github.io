Install & Quick Start

  1. Drop DirectorsCut.jar into plugins/ and restart/start server once.
  2. Confirm folders generate: plugins/DirectorsCut/cameras, sequences, qtes.
  3. Put YAML files into those folders (examples are in DirectorsCut/examples in this repository).
  4. Run /directorscut reload.
  5. Play a sequence: /directorscut play <sequenceId> [player].

Required/optional plugin stack

  • Required for core cameras/sequences: DirectorsCut only.
  • Required for mob instructions: MythicMobs (spawnmob, clearmobs).
  • Required for model bone cameras: ModelEngine (camspawn, camattach, camplay, camclear).
  • QTE: built-in (no external plugin needed).

Config highlights (config.yml)

debug:
  enabled: false
  verbose: false

model_engine:
  camera_rotation:
    yaw_offset: 0.0
    pitch_offset: 0.0

qte:
  indicator:
    defaults: ...
    type_overrides: ...

Use yaw_offset/pitch_offset if your ModelEngine camera bone orientation appears rotated from expected perspective.
