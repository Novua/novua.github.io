Plugin snapshot

Name: DirectorsCut
Command root: /directorscut
Paper API: 1.20.x (api-version: 1.20)
Soft dependency: MythicMobs
Cameras folder: plugins/DirectorsCut/cameras
Sequences folder: plugins/DirectorsCut/sequences

Sequence format versions (up-to-date)

Supported values are format: 1 and format: 2. If format is omitted or invalid, loading falls back to format 1.

Format Behavior Update notes When to use
1 glide(...) blocks script progression until the glide finishes. Legacy/default behavior kept for backward compatibility. Older sequences where timing depends on glide completion before next lines execute.
2 glide(...) no longer blocks; script can continue during movement. Introduced to support layered timing and richer cinematic pacing. New sequences that need title/command/effect overlap while camera movement continues.

Commands & permissions

Playback

  • /directorscut play <sequenceId> [player]
  • directorscut.admin.play
  • directorscut.play.other

Maintenance

  • /directorscut reload
  • /directorscut version / /directorscut update
  • /directorscut support
  • directorscut.reload
  • directorscut.version
  • directorscut.support

Camera helpers

  • /directorscut debugcamera <cameraId>
  • /directorscut testcamera <cameraId>
  • /directorscut editsequence makecamera
  • directorscut.debug, directorscut.sequence.edit

Sequence editor

  • /directorscut editsequence <sequenceId>
  • /directorscut editsequence preview|next|prev|delay|glide|save|cancel
  • directorscut.sequence.edit

Camera YAML schema

pack: optional_pack_id

cameras:
  camera_id:
    label: Human Readable Name
    anchor:
      type: player | entity | absolute | none
      source: self | interacted | targeter | mythicmobs_targeter | mythicmobs:<internalName>
      absolute:  # required when type: absolute
        world: world
        x: 0.0
        y: 64.0
        z: 0.0
        yaw: 0.0
        pitch: 0.0
    look_at:     # optional; defaults to anchor
      type: player | entity | absolute | none
      source: self
    positioning:
      mode: relative | anchored | absolute
      distance_x: 0.0
      distance_y: 1.5
      distance_z: -3.0
      # OR legacy style:
      distance: 4.0
      y_offset: 1.2
      yaw_offset: 180
      pitch: -10.0
      absolute:  # required when mode: absolute
        world: world
        x: 0.0
        y: 70.0
        z: 0.0
        yaw: 90.0
        pitch: -20.0
    lock_player:
      move: false
      look: false

Sequence YAML + script DSL

sequence: sequence_id
format: 1 | 2

settings:
  allow_exit: false
  freeze_player: false
  hide_gui: false

script:
  - static(camera_id)
  - delay 20
  - glide(from_camera,to_camera,60)
  - glide(from,to,80,curve=2.0,curve2=-1.0)
  - showtitle(10,60,10,"&6Title")
  - showsubtitle(10,60,10,"&7Subtitle")
  - showactionbar(0,40,10,"&fActionbar")
  - command(c="say hello", asPlayer=false, asOp=false)
  - spawnmob mob_internal_name
  - spawnmob(mob_internal_name)
  - spawnmob{mob=mob_internal_name;client=false}
  - spawnmob{mob=mob_internal_name;client=true}
  - clearmobs
  - conditionend
  - end

endconditions:
  - offhandswap
  - sneak
  - hotbarkey(0)

endscript:
  - command(c="say sequence ended")
