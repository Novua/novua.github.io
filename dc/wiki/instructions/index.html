Sequence Instruction Reference

InstructionWhat it doesExample
static(id) Instantly jump to a camera. static(village_intro)
glide(from,to,duration) Smoothly transition between cameras over a number of ticks. glide(intro1,intro2,80)
delay <ticks> Pause the script for a number of ticks (20 ticks = 1 second). delay 40
spawnmob <mythic_id> Spawn a MythicMobs mob and register it for camera targeting. spawnmob cinematic_dummy
command(c="...",asplayer=<bool>,asop=<bool>) Execute any Bukkit command. command(c="gamemode adventure %player%",asplayer=false)
showtitle(...) Show a title overlay (also showsubtitle, showactionbar). showtitle(10,40,10,"&eIncoming!")
clearmobs Despawns any MythicMobs created by spawnmob. clearmobs
end Immediately end the sequence. end
conditionend Terminate the current conditional branch and continue. conditionend

Need to stop a sequence early? See end conditions.
