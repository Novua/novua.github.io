Commands & Permissions

All admin commands are under /directorscut.

Command reference

CommandDescriptionPermission
/directorscut play <sequence> [player]Play a sequence for yourself or another player (admin workflow).directorscut.admin.play (+ directorscut.play.other for [player])
/directorscut reloadReload cameras and sequences from disk.directorscut.reload
/directorscut version or /directorscut updateCheck the latest published version from https://novua.net/dc/latest.txt.directorscut.version
/directorscut supportShows support Discord and a generated support ID (click to copy in chat).directorscut.support
/directorscut debugcamera <cameraId>Print camera resolution/debug info.directorscut.debug
/directorscut testcamera <cameraId>Preview a camera for 3 seconds.directorscut.debug
/directorscut editsequence ...Create, modify, preview, and save sequence scripts in-game.directorscut.sequence.edit

Support ID placeholder format

The support command uses predefined storefront placeholders and does not read editable config fields:

((USER_NAME))-((ORDER_UUID))-((USER))
