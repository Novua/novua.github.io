Commands & Permissions

CommandDescriptionPermission
/directorscut play <sequenceId> [player]Plays a sequence for self or a specified target player.directorscut.admin.play + directorscut.play.other for other target.
/directorscut reloadReloads camera, sequence, and QTE definitions from disk.directorscut.reload
/directorscut debugcamera <cameraId>Prints resolved world/xyz/yaw/pitch for current context.directorscut.debug
/directorscut testcamera <cameraId>3-second spectator preview through an invisible display entity.directorscut.debug
/directorscut editsequence ...In-game sequence editor session workflow.directorscut.sequence.edit
/directorscut supportShows Discord and clickable support ID.directorscut.support
/directorscut version / updateChecks latest DirectorsCut version asynchronously.directorscut.version
