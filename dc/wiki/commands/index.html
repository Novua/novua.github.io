Commands & Permissions

CommandDescriptionPermission
/directorscut play <sequence> [player]Play a sequence for yourself or another player.directorscut.play / directorscut.play.other
/directorscut reloadReload cameras and sequences from disk.directorscut.reload
/directorscut debugcamera <cameraId>Print camera resolution/debug info.directorscut.debug
/directorscut testcamera <cameraId>Preview a camera for 3 seconds.directorscut.debug
/directorscut editsequence create <sequenceId>Create a new sequence file in-game.directorscut.sequence.edit
/directorscut editsequence <sequenceId>Enter sequence editor mode for an existing sequence.directorscut.sequence.edit
/directorscut editsequence makecameraName a new camera in chat and place it at your current location.directorscut.sequence.edit
/directorscut editsequence addstatic|addglide|adddelay|addcommandAppend script instructions while editing.directorscut.sequence.edit
/directorscut editsequence removeRemove the currently selected instruction.directorscut.sequence.edit
/directorscut editsequence preview|next|prev|delay|glide|save|cancelCore sequence editor actions.directorscut.sequence.edit
