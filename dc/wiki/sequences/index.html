Sequence Format

Sequences live in plugins/DirectorsCut/sequences/*.yml. A file can define one sequence or many under sequences:.

sequence: intro_demo
format: 2
settings:
  allow_exit: false
  freeze_player: true
  hide_gui: false
  max_uses: 3
script:
  - static(intro_shot)
  - showtitle(10,40,10,"&6Welcome")
  - delay 20
  - qte(id=jump_open)
  - conditionend
endscript:
  - command(c="say Sequence ended",asplayer=false)
endconditions:
  - sneak
  - offhandswap
  - hotbarkey(8)

Formats

  • format: 1: glide blocks script advancement until transition ends.
  • format: 2: glide runs asynchronously while script continues.

Settings behavior

  • allow_exit: if false, sneak exit is blocked.
  • freeze_player: forces spectator/flight lock state for cinematic playback.
  • hide_gui: currently parsed; use with resource/UI strategy as needed.
  • max_uses/maxuses: per-player sequence use cap; <=0 is treated unlimited.
