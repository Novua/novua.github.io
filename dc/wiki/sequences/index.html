Create Sequences

Sequences are YAML files in sequences/*.yml that define a list of instructions to play cameras over time.

Sequence template

sequence: castle_intro

settings:
  allow_exit: false
  freeze_player: true
  hide_gui: true

script:
  - static(courtyard_wide)
  - delay 40
  - glide(courtyard_wide,gate_closeup,60)
  - showtitle(20,60,20,"&6Welcome to the keep")
  - end

endscript:
  - command(c="title %player% clear",asplayer=false)

Settings explained

  • allow_exit — Allow players to exit the sequence early (e.g., via sneak).
  • freeze_player — Prevent player movement during playback.
  • hide_gui — Hide the player HUD during the sequence.

For a full list of instruction lines, visit the instruction reference.
