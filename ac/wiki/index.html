Quick start checklist
  1. Drop AnimationsCore and MythicMobs into plugins/ then reboot the server to generate data folders.
  2. Install the Mythic packs referenced by the shipped YAML entries (joins, leaves, AFK, emotes, customs, and more).
  3. Set default cinematics inside config.yml, grant staff the animcore.admin.* suite, and reload.
  4. Open /animations in-game to validate the GUI layout and permissions before inviting players.

Server requirements

AnimationsCore targets modern Paper / Spigot stacks and leans on MythicMobs for playback.

  • Java 17 runtime (matching Paper / Spigot 1.19+).
  • MythicMobs installed and configured with the relevant Mythic IDs.
  • Network access for optional starter pack downloads on first boot.
Pro tip

Use disabled_worlds in config.yml to limit cinematics to lobby or hub servers while keeping survival worlds untouched.
